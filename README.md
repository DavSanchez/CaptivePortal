# Captive Portal
A **Node.js** &amp; **MediaStream Recording** Captive Portal Web App to use
with **CoovaChilli**

## What is this?
This is a captive portal service designed to serve as landing page and login process
for a CoovaChilli implementation. It uses Node.js as backend and the MediaStream
Recording API and CoovaChilli's JSON interface for client management.
